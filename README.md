# 🟦 CleanSort — Умный Сортировщик Загрузок

![Bash](https://shields.io)
![License](https://shields.io)

**CleanSort** — это стильный терминальный скрипт для Linux, который наводит порядок в папках `Downloads` (Загрузки) одним нажатием кнопки. Больше никакого хаоса — только структура и эстетика.

---

## ✨ Особенности

*   🌀 **Стильный ASCII-интерфейс** в монохромном голубом стиле.
*   🌍 **Мультиязычность**: автоматически определяет, как называются ваши папки (`Downloads` или `Загрузки`).
*   📂 **Умная сортировка**: раскидывает файлы по категориям:
    *   `Изображения` (jpg, png, svg...)
    *   `Документы` (pdf, docx, txt...)
    *   `Видео` (mp4, mkv, avi...)
    *   `Архивы` (zip, rar, 7z...) — создаются прямо внутри папки загрузок.
*   🚀 **Легкая установка**: запускается как системная команда.

## 🛠 Установка

Установите CleanSort одной командой прямо в систему:

```bash
curl -sSL https://githubusercontent.com | bash
```

*После установки просто введите `cleansort` в любом терминале.*

---

## 📦 Ручная установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Перейдите в папку и дайте права на запуск:
   ```bash
   cd cleansort && chmod +x cleansort
   ```
3. Запустите:
   ```bash
   ./cleansort
   ```
